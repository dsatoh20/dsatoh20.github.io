    論文読書メモ¶

    社会疫学の論文を読んだ記録。主に自分の学習記録として、出願時の資料としての活用も見据えて書く。

    • メモ本文（RQ・知見・考察など）: Github Pages公開を前提に書く
    • 批判的吟味 by Claude Code: 学習深化を目的とする。考察の不備（論理の飛躍・見落とし・深掘り不足）の指摘と、学術的な切り口での論文解釈を行う。専門用語・論理的厳密さを優先する

    ファイル命名規則¶

    YYYY-MM-DD_著者(年)_タイトル略称.md

    例: 2026-04-15_LinkPhelan1995_fundamental-causes.md

    ペーパータイプ別の構成¶

    メモの構成と批判的吟味の切り口は、ペーパータイプに応じて変える。

    実証研究（RCT・観察研究）¶

    • RQ: 何を問うているか
    • 手法: どうやって答えたか（PICO）
    • 知見: 何がわかったか
    • 考察: 自分の解釈・疑問・臨床との接続
    • 批判的吟味 by Claude Code: バイアス・交絡・統計的妥当性

    理論・概念論文¶

    • 主張: 著者が何を言いたいのか（テーゼ）
    • 背景: なぜこの主張が必要だったか
    • 論拠: どんな根拠・推論で主張を支えているか
    • 含意: 研究デザイン・政策・臨床への示唆
    • 考察: 自分の解釈・疑問・他論文との接続
    • 批判的吟味 by Claude Code: 論拠の強度・操作化可能性・競合理論との比較

    システマティックレビュー・メタアナリシス¶

    • RQ: 何を問うているか
    • 手法: サーチ戦略・採択基準・統計手法（メタアナリシスの場合）
    • 知見: 何がわかったか
    • 考察: 自分の解釈・疑問・臨床との接続
    • 批判的吟味 by Claude Code: 採択バイアス・異質性・エビデンスの総合的強度

    輪読会メモ¶

    >**臨床医のための疫学・統計 輪読会メモ**と記されている場合、「臨床医のための疫学・統計（磯ら, 朝倉書店）」をテキストとした輪読会のメモである。批判的吟味は社会疫学的視点よりも、研究デザイン・バイアス・統計手法の妥当性を中心とした臨床疫学的議論にフォーカスする。
