# Appendicectomy plus standard medical therapy versus standard medical therapy alone for maintenance of remission in ulcerative colitis (ACCURE): a pragmatic, open-label, international, randomised trial
10.1016/S2468-1253(25)00026-3
>**臨床医のための疫学・統計 輪読会メモ**

## RQ
潰瘍性大腸炎の治療法として、虫垂切除術は有効か。
## 手法
オランダ、アイルランド、UKの多施設合同ランダム化比較試験。
### P
18歳以上の潰瘍性大腸炎患者のうち、過去12か月以内に再発経験があり、現在は寛解している者、201名
### I
維持療法の継続＋腹腔鏡的虫垂切除術
### C
維持療法の継続
### O
虫垂切除群で、1年以内再発率が小さい
## 知見
- 虫垂切除術上乗せは、潰瘍性大腸炎の1年以内再発率を低下させる。
## 考察
- 手術は往々にしてリスクを伴うものである。本研究では、術後合併症のリスクと1年以内再発率低下のベネフィットを直接天秤にかけてはいないが、暗に前者を小さく見積もったものと思われる。これが妥当なものか、簡単には判断しかねるが、100件程度の腹腔鏡手術ではそうそう重篤な合併症を生じないという結果を鑑みるに、やはり再発予防に手術上乗せは有効と言えるだろう。
- 手術という介入を行う以上、オープンラベル、非盲検はしかたない。気になるのは、試験に参加した医療者たちが、虫垂切除にどれだけUC再発予防効果を期待したかという点。虫垂が免疫機能と関連するという報告は以前からあったものの、試験参加者が抱いた期待感はいかほどプラセボとして働いたかは議論してもよいだろう。一見手術のリスクのみを背負わされるような試験で、患者を納得させる方便として、腹腔鏡での虫垂切除術に重篤な合併症がほとんど起こらない、という前提は必要だったに違いない。
## 批判的吟味（疫学的視点）by Claude Code

### 1. 非盲検化による detection bias のリスク

あなたが指摘した「期待感＝プラセボ」は重要な論点だが、疫学的に整理すると **detection bias** として扱うのが正確。外科的介入である以上 allocation concealment の破綻は不可避だが、問題は primary outcome「再発」がどう定義・測定されたかによってリスクが変わる点にある。

- 再発が **内視鏡的所見**（例: Mayo スコア）で定義されていれば、判定医が割付けを知ることで判定が甘くなる（介入群に再発と診断しにくい）detection bias が入り得る。
- 再発が **患者の症状報告**（PRO）主体なら、期待感による performance bias の寄与が相対的に大きくなる。
- 論文の supplementary に掲載されているはずの adjudication（転帰判定）のブラインド手続きを確認することが批判的吟味の第一歩。

### 2. Surgical placebo effect の問題

外科的介入の RCT では **sham surgery 対照**が理論上の gold standard（例: 半月板手術の Sham-Surgery RCT, Sihvonen 2013, NEJM）。しかし本試験では倫理・実施可能性から採用不可。これは単なる limitation ではなく、「虫垂切除術のベネフィットの一部は surgical placebo effect によるものかもしれない」という解釈を完全に排除できない構造的問題として残る。オープンラベルを「しかたない」と処理するだけでなく、このエビデンスの限界を明示的に議論する価値がある。

### 3. Pragmatic デザインの内的・外的妥当性のトレードオフ

本試験は "pragmatic" と明示されている。Pragmatic RCT は外的妥当性を優先するため、実施条件（維持療法の内容）が施設間・患者間で不均一になりやすい。

- **介入の異質性**：維持療法は施設・担当医によって異なる可能性があり、これが効果推定値の精度を下げる交絡になりうる。
- **サブグループ解析**の必要性と多重比較の問題：「どの維持療法下で効果が大きいか」という問いへの答えが本試験では不十分な場合、次の研究課題として残る。

### 4. 患者選択基準が作り出す Selection bias

P の定義「過去12か月以内に再発あり、現在は寛解」は **relapsing-remitting フェノタイプ** を選んでいる。これは、最も外科的介入を試みたいグループ（難治性・頻回再燃例）ではなく、比較的軽症〜中等症の UC フェノタイプに限られる。

- この選択基準は内的妥当性のために適切だが、日常臨床で虫垂切除を検討する患者層（重症・生物学的製剤無効例）と乖離する可能性があり、**外的妥当性（一般化可能性）**への留保が必要。

### 5. フォローアップ期間と統計設計

1年という追跡期間は、UC のような慢性疾患の治療効果評価としては短め。

- 仮に虫垂切除の効果が免疫学的リモデリングを介するなら、効果発現に時差が生じる可能性がある一方、短期的な外科的侵襲によるストレス応答で一時的に腸管炎症が抑制されているだけかもしれない。
- 検出力の問題（sample size 201 が primary outcome の再発率の推定値の精度にどう影響するか）を評価するには、プロトコール論文または Methods に記された α・power 設定の確認が必要。

### 6. ITT 解析の徹底度

Pragmatic RCT では per-protocol 解析より **intention-to-treat（ITT）解析**が原則。しかし手術群で「手術を拒否した」「術後合併症で維持療法を変更した」ケースの取り扱いが ITT を歪める可能性がある。CONSORT フローダイアグラムを確認すべき点。

---
**全体評価**：RCT というデザイン上の強みはあるが、非盲検化・surgical placebo effect の残留・pragmatic 設計による異質性が内的妥当性を制約する。「手術は有効か」に対して Yes と言えるが、「何が効いているか（真の生物学的効果 vs 手術プラセボ vs コンコミタントな診療強化）」は本試験では分離できない。この点を押さえておくと、この結果の解釈が深まる。
