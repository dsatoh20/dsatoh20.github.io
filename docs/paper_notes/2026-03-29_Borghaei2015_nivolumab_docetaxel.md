# Nivolumab versus Docetaxel in Advanced Nonsquamous Non–Small-Cell Lung Cancer
DOI: 10.1056/NEJMoa1507643
>**臨床医のための疫学・統計 輪読会メモ**

## RQ
進行非扁平上皮・非小細胞肺癌（NSCLC）に対する二次治療として、ドセタキセルに比べてニボルマブは全生存期間を延長させるか。
## 手法
国際共同第3相ランダム化比較試験。
### P
stage IIIB以上または再発のNSCLC、かつ一次治療経過中・後に進行または再発のあったパフォーマンスステータス1以下の18歳以上患者、582名。
### I
ニボルマブ投薬
### C
ドセタキセル投薬
### O
全生存期間、腫瘍縮小効果、無増悪生存期間、PD-L1発現、有害事象の発生頻度

## 知見
- ドセタキセルに比してニボルマブは全生存期間を延長させる。
- ドセタキセルに比してニボルマブはgrade3,4の治療関連副作用の発生頻度が小さい。
## 考察
- prior regimensとline on therapyについて、stratification factorsと表現されているが、層別化してそれぞれの層で結果を示しているわけではない。だから、stratification factorsというよりは、counfoundingと表現するほうが相応しいのではないか。
- Figure2は単変量解析解析。その意味で、unstratifiedという語を用いている模様だが、むしろこれを層別化と呼ぶべきではなかろうか。
- Figure2はサンプル数が少ないものもあり、統計学的なメッセージ性はやや劣る。しかし、ニボルマブ適用範囲の議論をする上で、臨床医としては欲しい情報であろうから、あえて掲載したと考えられる。
- Figure2にて、当初PS1だったがPS3となったサンプルは、intention-to-treatで扱ったとある。ランダム化による交絡制御効果を損なわない意図がみえる。
- PD-L1発現が高い治療効果と相関することに言及する一方で、PD-L1発現を問わず適応できると主張している。[肺がん診療ガイドライン2025年版](https://www.haigan.gr.jp/publication/guideline/examination/2025/1/2/250102070100.html#7-2)では、PD-L1 Tumor Propotion Score 50%をカットオフに、治療方針を示す樹形図が2種類示されている。また、この最新ガイドラインでは、本研究のDiscussionにて今後研究が期待されるとして触れられた、combination therapyがもはや標準治療となっていることが確認できた。
- 本研究の限界として、PS不良例での適応の議論に用いることができないことが挙げられる。[肺がん診療ガイドライン2025年版](https://www.haigan.gr.jp/publication/guideline/examination/2025/1/2/250102070100.html#cq68)でも指摘されている。

## 批判的吟味（疫学的視点）by Claude Code

### 研究デザインの妥当性

国際共同第3相RCTとして、ランダム化・並行群間デザインによる内的妥当性は高い。ただし**オープンラベル**（非盲検）であるため、患者・医師が投与薬を知っている。OSは死亡という客観的エンドポイントで盲検化の影響を受けにくいが、有害事象の主観的評価やdocetaxel群への後治療（immunotherapy）の選択が結果に影響した可能性に留意が必要。

### Stratification factorsの概念整理

ユーザーの疑問「stratification factorsよりconfoundingでは？」は重要な問いだが、RCTの文脈では用語の意味が異なる。

- RCTの**stratified randomization**：PD-L1発現・prior regimensなどの因子ごとに層を分け、層内でランダム化することで、これらの変数が介入群・対照群間で均等になるよう保証する操作。「交絡を除去する」というよりは**交絡が生じないようにデザインで担保**する手段。
- 観察研究の**confounding**：因果推論において交絡因子を特定・調整する必要があるという概念。RCTではランダム化によりそもそも交絡が制御されているため、この語を使うのは文脈的にミスマッチ。
- **層別化された変数はCox回帰でcovariateとして投入**されることが多く（調整HR算出のため）、これが「stratification factorsを解析で使っている」部分。

→ stratification factorsという表現はRCTとして正確。

### Figure 2の「unstratified」について

ユーザーが「層別化と呼ぶべきでは」と感じたのは理解できる表現の揺れだが、指示対象が異なる。

- **KM曲線をサブグループごとに描く**こと → サブグループ解析（subgroup analysis）
- **「unstratified log-rank test」** → 各サブグループ内のlog-rank検定において、他の共変量による層別補正を行っていないという意味。すなわちsimple log-rank test。

「unstratified」はKM曲線の描き方ではなく、検定の修飾語。ユーザーの観察は正しく、実態としては単変量のサブグループ解析。

### サブグループ解析（Figure 2）の解釈上の注意

- サンプルサイズが小さいサブグループでは信頼区間が広く、点推定の信頼性が低い（ユーザーの指摘通り）。
- **多重比較問題**：複数サブグループを検定すれば偶然有意になる確率が上がる。Bonferroni補正等は行われていない。
- Pre-specified（事前設定）かpost-hoc（事後探索）かの明示が弱く、exploratory（仮説生成的）解析として読むのが疫学的に適切。
- ユーザーが指摘した「臨床医には欲しい情報」というのはその通りで、Figure 2は意思決定支援的なメッセージを持つが、統計的確証を与えるものではない。

### ITT解析について

PS1→PS3への変化例をITTで扱うという記述に注目したのは適切。

- **ITT（intention-to-treat）解析**は「割り付けられた通りに分析する」原則で、プロトコル逸脱例も含めることで**ランダム化による交絡制御の恩恵を保持**する。
- Per-protocol解析に切り替えると、「プロトコルを完遂できた人」という選択バイアスが生じる。

### 外的妥当性

PS 0–1のみの対象（PS ≧ 2を除外）、stage IIIB以上または再発例に限定されており、これが外挿の主な制限。ユーザーが参照したガイドラインでの指摘と一致する。
