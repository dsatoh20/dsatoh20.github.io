# Predicting tumor response and prognosis to neoadjuvant chemotherapy in esophageal squamous cell carcinoma patients using PERCIST: a multicenter study in Japan
DOI: 10.1016/j.cell.2015.11.001
>**臨床医のための疫学・統計 輪読会メモ**

## RQ
食道がん術前化学療法の予後評価にwhole-PERCISTを用いるべきか。
## 手法
### P

### E
whole-PERCIST
### C
原発巣PERCIST、病理組織的像、…
### O
予後指標としての有用性

## 知見
- whole-PERCISTはNACの全生存期間の予測に有用。
- リスク分類のカットオフ値はSULpeak response = -50%とすべき。

## 考察
- 国内初の多施設共同研究であることがこの報告のバリューの一つになっている。
- 機械学習の古典、決定木から派生した、Survival tree modelを用いた点が面白い。生存時間解析としてスタンダードな、Cox比例ハザードモデルに対する優位点として、
    1. 比例ハザード性を仮定しなくてよい
    2. 多重共線性を考慮しなくてよい
    3. 強いinteractionを自動て発見できる
    4. 非線形な関係を扱える
    5. 欠損値を補完しなくてよい

    が挙げられる。限界としては、末端ノードほどサンプル数が小さくなり、結果の信頼性に乏しくなっていくことが挙げられる。
- Survival tree modelは、患者リスク群の分類を行ったが、これは、予後不良群にはさらなる早期介入をしたいというモチベーションによるものと考えられる。
- がんの治療法を検討する研究のプライマリエンドポイントとしては、Progression Free Survivalあるいは腫瘍縮小効果が設定される。本例では、まず後者をPETで判定し、その後それが本当に前者: PFSに関連するかを議論している。
## 批判的吟味（疫学的視点）by Claude Code

### 研究デザインと内的妥当性

**後ろ向き多施設観察研究**であり、介入割付がないため交絡の制御が課題となる。

- **選択バイアス**: NAC適用患者に限定されており、外科的切除単独・放射線化学療法を受けた患者は含まれない。NAC適応の判断は施設・術者の裁量に依存するため、より良好な全身状態の患者が偏って選ばれている可能性がある（適応症バイアス）。
- **測定バイアス**: 7施設で異なるPETスキャナー・プロトコルが使用されたはず。SULpeakの施設間ばらつきがwhole-PERCISTの評価に与える影響が方法論として十分議論されているか確認が必要。
- **交絡**: ステージ、リンパ節転移数、術前の栄養状態（食道がんでは重要）、化学療法レジメン（CF vs DCF等）といった予後規定因子が交絡になり得る。

### 統計手法の選択

Survival tree modelの選択は理由として挙げられた5点（比例ハザード性不要・多重共線性不要など）は正当だが、**過適合（overfitting）リスク**が懸念される。

- 177例を7施設から集積しており、施設あたり平均25例程度。Survival treeの分岐ごとにサンプルが減少するため、末端ノードでは少数例に基づいた予測になる。
- **カットオフ値（SULpeak response = −50%）がこのデータから導出されている**点は重大で、同じデータで導いたカットオフを同じデータで評価しているため、内部評価の精度指標は楽観的に偏る。外部コホートでの検証なしにはカットオフの汎用性は保証できない。

### 外的妥当性（一般化可能性）

- **日本の7施設・食道扁平上皮癌（SCC）のみ**。欧米では食道腺癌が主流であり、本知見の地理的・組織型的一般化は限定的。
- 国内初の多施設研究という点は強みだが、施設間の患者特性・治療プロトコルの均質性についての情報が必要。

### 臨床的意義と今後の課題

- プライマリエンドポイントをPES（病理組織的効果）でなくOSに置いたことは**患者中心アウトカム**として適切。
- ただし、whole-PERCISTが高リスクと分類された患者に対して何を変えるのか（追加治療？緩和ケアへの移行？）という**意思決定への接続**が示されていないと、臨床ツールとしての価値は限定的。
- 今後の課題として、①外部コホートでの検証、②adenocarcinomaへの適用可能性、③マルチモーダルな画像情報（CT, MRI）との比較が挙げられる。
